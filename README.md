## Download it locally or for pull requests

You can load it in there if you have [JetBrains Webstorm](https://www.jetbrains.com/webstorm/).

### For other code editors (such as VSCode)

#### Linux

```bash
sudo apt update && sudo apt install git
sudo apt install nodejs
git clone https://github.com/notmithun/notmithun.github.io.git
cd "notmithun.github.io"
npm install
```

#### Windows

```
winget install --id Git.Git -e --source winget
winget install OpenJS.NodeJS.LTS
git clone https://github.com/notmithun/notmithun.github.io.git
cd "notmithun.github.io"
npm install
```

#### MacOS

*I do not own a Macbook or Mac, so if anything goes wrong please tell me what I did wrong and maybe put it in the "Issues" in GitHub*

**You must get Homebrew before following.**

```bash
brew install node
brew install git
git clone https://github.com/notmithun/notmithun.github.io.git
cd "notmithun.github.io"
npm install
```

Also, if you're getting an error like npm or git not found, I'd recommend you restart your then continue.
